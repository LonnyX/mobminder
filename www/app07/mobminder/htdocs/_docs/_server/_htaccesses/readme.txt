
subdomains are re-routed to folders located under /srv/web/be.mobminder.com

If you want to organise the url re-writing for a given subdomain, open de .htaccess file located in the sub-domain corresponding folder. 

If you want to organise the routing of a given subdomains to a given folder (e.g. add a new sub-domain like foo.mobminder.com), then you need to adapt the DNS.
